## 計算結果摘要

- **計算區間**：500–600 m（100 m，資料完整率 100%）
- **Q1 Vshale**：Linear GR 法，GR_min=60 / GR_max=135 gAPI，平均 Vsh=0.57
- **Q2 PHIE**：密度孔隙度法，RHOMA=2.65 / RHOF=1.05 g/cm³，平均 PHIE=0.145
- **Q3 Sw**：Archie 公式，Rw=1.57 ohm·m（Rwa 法），a=1 m=2 n=2，平均 Sw=0.97
- **Q4 潛力砂層**：530–535 m、556–557 m（Vsh<0.35, PHIE>0.08, Sw<0.75）
